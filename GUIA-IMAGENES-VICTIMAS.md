# Guía de Imágenes de Víctimas - Casos Reales

Esta guía documenta las fuentes oficiales y confiables para obtener fotografías respetuosas de las víctimas incluidas en el proyecto.

## Principios Éticos

1. **Solo fotos en vida** - Nunca usar fotos post-mortem o de escenas del crimen
2. **Fuentes verificadas** - Solo medios reconocidos y fuentes oficiales
3. **Respeto a las familias** - Usar fotos autorizadas cuando sea posible
4. **Atribución clara** - Siempre documentar la fuente original

---

## Estructura de Carpetas

```
public/
└── images/
    ├── cases/          ← Imágenes de portada de casos
    └── victims/        ← Fotos de víctimas (crear esta carpeta)
```

---

## Comandos Disponibles

```bash
# Actualizar la base de datos con las URLs e imageSource
npx tsx prisma/update-victim-images.ts

# Ver solo la guía de descarga
npx tsx prisma/update-victim-images.ts --guide
```

## Proceso de Descarga

1. **Visitar la fuente** indicada en la tabla
2. **Buscar la foto** más respetuosa (retratos, fotos familiares)
3. **Descargar** la imagen
4. **Renombrar** con el nombre de archivo indicado
5. **Guardar** en `public/images/victims/`
6. **Ejecutar** `npx tsx prisma/update-victim-images.ts`

---

## Consideraciones Legales

- Las fotos del **FBI** y otras agencias gubernamentales de EEUU son de **dominio público**
- Las fotos de **medios de comunicación** generalmente están protegidas por derechos de autor
- Para uso en un sitio web público, considera:
  - Contactar a los medios para obtener permiso
  - Usar solo fotos claramente de dominio público
  - Enlazar a fuentes en lugar de alojar las imágenes

---

## Contacto con Familias

Para casos recientes, la opción más ética es:
1. Contactar a las familias o sus fundaciones
2. Solicitar permiso para usar fotografías específicas
3. Dar crédito apropiado

**Fundaciones y sitios oficiales de familias:**
- Madeleine McCann: findmadeleine.com
- Maura Murray: mauramurraymissing.org
- Natalee Holloway: Natalee Holloway Resource Center
